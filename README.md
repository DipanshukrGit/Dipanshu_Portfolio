## 🌟 Overview

**`Dipanshu_Portfolio`** is my personal, fully responsive, animated portfolio website — built to showcase who I am, what I do, and how you can reach me. It combines performance, beautiful UI, and smooth animations to create a standout web presence.

Tech stack includes:

- ⚡ **Next.js** – for fast, modern React-based development
- 🎨 **Tailwind CSS** – for a clean and fully responsive layout
- 🧿 **GSAP** – for buttery smooth animations on scroll and load
- 🧩 **Shadcn UI** – for styled components that fit modern design systems
